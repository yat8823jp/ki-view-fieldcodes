# kintone フィールドコード表示

kintone で現在表示しているフィールドのフィールドコードを画面に表示・非表示切り替える Chrome 拡張機能です。表示されたチップをコピーすればクリップボードにコピーできます。

## バージョン

### v1.2.0（2025年3月）

- **レコード編集画面に対応**: レコード編集・追加画面でもフィールドコードを表示できるようになりました（従来は詳細・一覧・印刷画面のみ対応）
- 管理画面（アプリの設定など）では動作しない仕様です。レコードの詳細・一覧・編集画面でのみご利用ください

### v1.0.0

- 初回リリース（レコード詳細・一覧・印刷画面対応）

## 機能

- **レコード詳細画面**: 各フィールドにフィールドコードをバッジ表示
- **レコード編集・追加画面**: 同上
- **レコード印刷画面**: 同上
- **レコード一覧画面**: 表形式の一覧で各列にフィールドコードを表示

## インストール方法

1. Chrome で `chrome://extensions/` を開く
2. 「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダ（`vew-fieldcodes`）を選択

## 使い方

1. kintone のアプリ画面（レコード詳細・一覧・編集など）を開く
2. ブラウザのツールバーにある拡張機能アイコンをクリック
3. 「表示する」のトグルをオンにするとフィールドコードが表示されます
4. オフにすると非表示になります

設定は保存されるため、次回 kintone を開いたときも同じ状態で表示されます。

## 対応環境

- Chrome（Manifest V3 対応）
- kintone（cybozu.com / cybozu.biz / kintone.com）

## 注意事項

- テーブル内のフィールド、関連レコード、ステータス、作業者など、一部のフィールドは kintone API の制限により取得できない場合があります
- kintone のアップデートにより、表示が変わる可能性があります
